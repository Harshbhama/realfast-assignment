# Claims Processing System - Frontend

React + Vite frontend for the Claims Processing System backend.

## Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on http://localhost:5173 and proxies `/api` requests to http://localhost:3000.

## Prerequisites

The backend must be running on port 3000:

```bash
cd backend
npm start
```

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Router
